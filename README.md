# SwitchOn

A premium, production-ready, fully responsive startup website for SwitchOn built with static HTML, CSS, and vanilla JavaScript.

## Deploy on GitHub Pages

1. Push this folder to a GitHub repository.
2. Open the repository settings.
3. Enable GitHub Pages and select the main branch root folder.
4. The site will be available at your GitHub Pages URL.

## Notes

- No backend server is required.
- Firebase form submissions use placeholder configuration and will fall back gracefully when not configured.
- The site includes SEO metadata, PWA support, a service worker, and dark/light theme support.
